-- SUMMARY --

Accordion Hierarchy creates multilevel JQuery UI accordions with header and body areas for each item.
Originally based on the Hierarchical Text module at http://drupal.org/project/text_hierarchical


Images from https://www.iconfinder.com/iconsets/faticons, free for commercial use.

-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see https://drupal.org/node/895232 for further information.
